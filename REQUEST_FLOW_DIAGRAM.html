用户请求流程图 — 从前端到后端

sequenceDiagram participant U as 用户 participant FE as 前端 (script.js) participant API as FastAPI (app.py) participant RAG as RAG系统 (rag_system.py) participant AI as AI生成器 (ai_generator.py) participant LLM as LLM API participant Tool as 搜索工具 (search_tools.py) participant VS as 向量存储 (vector_store.py) participant DB as ChromaDB U->>FE: 输入 "What is MCP?" FE->>FE: 清空输入框，显示loading FE->>API: POST /api/query
{query, session_id} API->>API: 解析请求 (QueryRequest) API->>RAG: rag_system.query(query, session_id) RAG->>RAG: 构造提示词 RAG->>RAG: 获取对话历史 RAG->>AI: generate_response(query, history, tools) rect rgb(30, 41, 59) Note over AI,LLM: 第1次 API 调用 — 决定是否搜索 AI->>LLM: Messages API (带tools) LLM-->>AI: stop_reason: tool_use
调用 search_course_content end AI->>Tool: execute("search_course_content", query="MCP") Tool->>VS: search(query="MCP") VS->>DB: course_content 语义检索 (top 5) DB-->>VS: 5个文本块 + 元数据 VS-->>Tool: SearchResults Tool->>VS: get_lesson_link() / get_course_link() VS-->>Tool: 课程视频 URL Tool->>Tool: _format_results() 存入 last_sources Tool-->>AI: 格式化的搜索结果文本 rect rgb(30, 58, 95) Note over AI,LLM: 第2次 API 调用 — 综合回答 AI->>LLM: Messages API (不带tools)
携带搜索结果 LLM-->>AI: 最终回答文本 end AI-->>RAG: response 文本 RAG->>RAG: 获取 sources (带链接) RAG->>RAG: 记录对话历史 RAG-->>API: (answer, sources) API-->>FE: {answer, sources, session_id} FE->>FE: marked.parse() 渲染Markdown FE->>FE: 渲染Sources为可点击链接 FE-->>U: 显示回答 + 来源链接
